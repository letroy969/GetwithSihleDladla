## Stack

| Layer | Tech |
|-------|------|
| Frontend | React 18, Vite, Three.js |
| UI | Lucide React, custom CSS animations |
| 3D | Three.js particle sphere |
| Deploy | GitHub Pages via GitHub Actions |

## Run Locally

```bash
npm install
npm run dev
```

## Deploy

Push to `main` — GitHub Actions builds and deploys automatically to GitHub Pages.

> Make sure GitHub Pages is set to deploy from **GitHub Actions** in your repo settings.
